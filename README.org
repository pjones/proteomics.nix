#+title: Proteomics.nix

A [[https://nixos.org/][Nix Flake]] for easy access to proteomics data analysis tools such as
[[https://openms.de/][OpenMS]].

* Installing Packages

  1. [[https://nixos.org/download/#download-nix][Install Nix]]

  2. Install the desired package:

     Note: Replace =PACKAGE= with the name of the desired package.  See
     below for a list of available packages.

     #+begin_src sh
       nix \
         --extra-experimental-features nix-command \
         --extra-experimental-features flakes \
         profile install github:pjones/proteomics.nix#PACKAGE
     #+end_src
